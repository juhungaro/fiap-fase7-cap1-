🧠 O Cérebro (Fase 6)

A inteligência preditiva foi construída e consolidada na fase anterior através da modelagem estatística de predição de risco e a colaboração inteligente entre agentes autônomos.

Foco: Captura física por firmware ESP32 em MicroPython, gateway FastAPI unificador, e layouts responsivos Web (Vercel) e Mobile (EAS Build APK).

IoT & Hardware

Captura Fisiológica em MicroPython

  • 🎛️
    Transição de Linguagem
    Migração do código do microcontrolador C/C++ (Arduino) para MicroPython estruturado no ESP32.
  • 🖥️
    Display Local OLED SSD1306
    Interface local do leito exibindo batimentos (BPM), saturação de oxigênio (SpO2) e temperatura corporal.
  • 🚨
    Sinalizadores Físicos (LEDs)
    Alertas visuais imediatos por GPIO: LED verde para estabilidade, LED vermelho piscando para ritmos críticos.
  • 🌐
    Transmissão WiFi urequests
    Envio periódico de payloads estruturados em JSON via requisição HTTP POST ao backend exposto.

IA & Governança

Motores de IA e Governança Clínica

  • 🔮
    Previsão Instantânea de ML
    Modelo clássico RandomForest (.joblib) calcula a probabilidade matemática de pico de risco a cada leitura.
  • 🤝
    Orquestração Multiagente
    Uso do OpenAI Agents SDK dividindo a triagem: Orquestrador -> Analista (ML) -> Especialista (Protocolos).
  • 🛡️
    Validador de Coerência
    Camada de regras de segurança médica que audita e bloqueia alucinações e incompatibilidades de classificação de IA.
  • 📝
    Logs de Auditoria JSONL
    Persistência estruturada em logs/decisoes_cardio_ia.jsonl mapeando o run_id, timestamp e validação de governança.

Distribuição

Deploys e Distribuição Profissional

🌐 Web: Deploy Vercel (CI/CD)

* **CI/CD Integrado:** Conectado ao repositório do GitHub, disparando novos builds automaticamente a cada commit na branch main.

* **Roteamento SPA:** vercel.json configurado com regras de reescrita para index.html para evitar erros 404 em rotas internas do React.

📱 Mobile: Expo EAS Build (APK)

* **Build na Nuvem:** Configuração do eas.json no perfil preview permitindo compilar o app nativo do Android em segundo plano sem SDK local.

* **Distribuição APK:** Geração do executável .apk direto em nuvem, permitindo a instalação via escaneamento do QR Code.
